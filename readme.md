# 要件定義書：SNS（X）プロフィールバナー画像生成ツール

## 1. プロジェクト概要
ユーザーが簡単にX（旧Twitter）のプロフィール上部に表示するバナー画像を作成できるWebツールを開発する。  
直感的な操作で背景・文字・装飾を設定し、リアルタイムプレビューを確認しながらPNG形式でダウンロードできるようにする。

---

## 2. 対象ユーザー
- SNSを利用する一般ユーザー
- 企業・個人事業主（ブランド用バナー）
- ゲーム配信者・クリエイター
- フリーランスの自己PR用途

---

## 3. 目的
- 簡単かつ即時にSNSバナーを作成可能にする
- デザインスキル不要で魅力的なバナーを生成
- ダウンロード後すぐにXへアップロードできる

---

## 4. 機能要件

### 4.1 必須機能（MVP）
1. **キャンバス生成**
   - 固定サイズ：1500×500px（X推奨サイズ）
   - 背景色・グラデーション・画像設定可能
2. **テキスト追加**
   - 任意のテキスト入力
   - フォント選択（Google Fonts対応、日本語可）
   - サイズ・色・位置の変更
3. **リアルタイムプレビュー**
   - 設定変更時に即時反映
4. **ダウンロード機能**
   - PNG形式で保存
   - 高解像度対応

### 4.2 追加機能（バージョンアップ時）
1. **テンプレート機能**
   - ビジネス、カジュアル、イベント用など複数デザインを用意
2. **AIキャッチコピー生成**
   - プロフィールやキーワードを入力 → AIがキャッチコピー提案
3. **SNSアイコン位置ガイド**
   - プロフィール画像が重なる部分を透過枠で表示
4. **季節イベントテンプレ**
   - 桜、海、紅葉、雪などの背景素材
5. **背景AI生成**
   - OpenAI / Stability AI を使った自動背景生成

---

## 5. 非機能要件
- **レスポンシブ対応**：PC・タブレット・スマホで利用可能
- **高速処理**：画像生成は2秒以内
- **ブラウザ上完結**：サーバー不要（可能な限りフロントエンド完結）
- **無料で利用可能**（初期段階）

---

## 6. 技術スタック案
- **フロントエンド**：React + Tailwind CSS
- **描画処理**：Fabric.js または Konva.js（Canvas操作用）
- **フォント**：Google Fonts API
- **ホスティング**：Vercel
- **（オプション）AI生成**：OpenAI API / Stability AI API
